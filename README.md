## 项目简介
### 基于计算机视觉的手语实时传译系统 (大一年度项目)
针对于聋哑人与健全人交流不便的问题

我尝试做出这样一种系统，实时将手语翻译为语音，也能将语音翻译为手语动画

我在努力中
